======================================
*Sola scriptura* ou *l’Écriture seule*
======================================

Quelle est la source? *Sola scriptura* porte sur la référence théologique à laquelle se fier. L’Écriture est l’autorité théologique ultime dans le protestantisme. C’est pourquoi c’est dans les témoignages humains relatés par la Bible que se trouvent la Parole de Dieu qui doit être toujours interprétée.
